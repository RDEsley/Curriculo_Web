# 📄 Curriculo_Web

Este repositório contém o meu currículo profissional em formato web, desenvolvido para ser uma vitrine interativa das minhas habilidades, formação acadêmica e trajetória na área de tecnologia. O projeto foi projetado para ser responsivo, moderno e de fácil navegação.

---

## ✨ Recursos

* **Interface Interativa:** Uso de JavaScript para alternância de temas e exibição dinâmica de conteúdos.
* **Acesso a Certificações:** Seção dedicada para visualização rápida dos cursos e certificados conquistados.
* **Design Responsivo:** Layout adaptável para dispositivos móveis e desktops utilizando CSS3.
* **Conectividade:** Links diretos para redes profissionais, GitHub e contato via e-mail.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização visual, responsividade e variáveis para temas.
* **JavaScript:** Lógica de interatividade e manipulação do DOM.
* **Google Fonts:** Tipografia moderna utilizando a fonte Roboto.

## 🚀 Como Executar o Projeto

### Pré-requisitos
* Um navegador web moderno (Chrome, Firefox, Edge, etc.).

### Passo a Passo

1. **Clone o repositório:**
   git clone https://github.com/RDEsley/Curriculo_Richard.git

2. **Acesse a pasta:**
   cd Curriculo_Richard

3. **Abra o arquivo:**
   Basta abrir o arquivo `index.html` no seu navegador de preferência.
