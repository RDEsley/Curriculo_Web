## 📌 Sobre o Projeto

Este projeto consiste em um **currículo profissional desenvolvido em formato de site**, criado para apresentar de forma moderna e interativa informações sobre minha trajetória acadêmica, habilidades técnicas e certificações.

A ideia é transformar o currículo tradicional em uma **experiência digital mais dinâmica**, facilitando a navegação e permitindo acesso rápido às minhas informações profissionais e projetos.

> 🎯 Foco em design moderno, responsividade e apresentação profissional de perfil na área de tecnologia.

## 🚀 Como Executar

### Pré-requisitos

Antes de começar, você precisa apenas de:

- Navegador moderno (Chrome, Edge, Firefox, etc.)
- Git instalado (opcional)

### Passo a Passo

```bash
# 1. Clone o repositório
git clone https://github.com/RDEsley/Curriculo_Web.git

# 2. Acesse a pasta do projeto
cd Curriculo_Web
```

3. Abra o arquivo **index.html** no navegador
